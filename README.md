# Interpreting Economic Evaluation Results in HTA

Workshop materials for the session on interpretation of economic evaluation results in Health Technology Assessment, delivered at IGGMC Nagpur (Feb 24–25, 2026).

Built with [Quarto](https://quarto.org). Deployed via GitHub Pages.

## Contents

- **Interactive Session** — ICER calculator, CE plane, CEAC, tornado diagram, EVPI, budget impact, case study
- **Presentation Slides** — Reveal.js slide deck
- **Practice Cases** — Indian HTA case studies with solutions
- **Resources** — Reference links and recommended reading

## Deployment

Push to `main` triggers automated rendering and deployment via `.github/workflows/publish.yml`.
